Click the buttons below to download the required icon files for your Chrome extension.

16x16


48x48


128x128


Instructions

  1. Click "Download All Icons" or download each icon individually
  2. Save the files in your extension folder (same directory as manifest.json)
  3. Make sure the files are named exactly:
    • icon16.png
    • icon48.png
    • icon128.png
  4. The icons feature a purple gradient with "CS" text (Collabstr Scraper)
